CROSSWORD HUNT

A Technical Treasure Hunt!!



Rules



ELIGIBILITY : This event is open for all the departments.

EVENT DESCRIPTION:

A Crossword hunt is a technical treasure hunt in which teams hunt for answers to puzzles at different places.

Round 1 :
TechQuest- Technical word search
In this round, each team will be given a word search puzzle of technical words. In the given time limit the team which finds all the technical words will go to the next round.

Round 2 :
Treasure Trove
In this round, each team will be given a crossword. For each word, teams will hunt for hints at different places. Solve the hints and find the answers to the crossword.
